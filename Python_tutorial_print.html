Pythonで出力：print文¶

  • コマンド画面に文字や値を表示させることを「出力する」という。
  • Pythonでは、プログラムの計算結果をコマンド画面上に表示させるためにprint()を用いる。
    • print('文字列')では、''で囲まれた部分がそのまま文字列とみなして画面に出力する。
    • print(変数)では，変数の値を画面に出力する。変数は文字列でも数値でも良い。
    • 他にも，print('文字列', 変数)やprint(f'文字列 {変数}')などのように便利な書き方がある。
  1. 文字列の表示

出力させたい文字列を''で囲み、print('文字列')とすると、コマンド画面に文字を表示させることができます。

In [36]:
print('hello world!')

hello world!

※''の値を変えて、ShiftキーとEnterキーを同時に押してみましょう。その下に表示された文字が変化するはずです。


また、カッコの中に変数を入れると、変数の値を出力させることができます。

In [37]:
text = 'abcde'
print(text)

abcde

※変数textの右辺の文字を変えて、ShiftキーとEnterキーを同時に押してみましょう。その下に表示された文字が変化するはずです。


文字列と変数は,を用いて横に並べることもできます。これまでの例と同様に、文字や変数の値を変えてみましょう。

In [38]:
text = 'abcde'
print('変数textの値は ', text, 'です.')

変数textの値は  abcde です.

変数の値が数値の場合には、その値を出力します。

In [40]:
x = 3.1415926
print('pi=', x)

pi= 3.1415926


print(f'文字列... {変数} 文字列... {変数}')では、文字列と変数の値を一緒に混ぜた形式で出力することができます。

In [42]:
pi = 3.14
e = 2.73
print(f'pi= {pi}, e= {e}')

pi= 3.14, e= 2.73
